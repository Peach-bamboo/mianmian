# ADR-003：使用轻量 Markdown 解析器

- 状态：已采用
- 日期：2026-06

## 背景

题库答案需要标题、列表、代码块、表格、图片和复制能力，但微信小程序不具备完整浏览器 DOM，项目也需要控制包体积。

## 决策

使用纯函数将 Markdown 解析为结构化 block，再用小程序原生组件渲染。

支持：

- 标题
- 段落
- 引用
- 列表
- 代码块与复制
- 加粗和行内代码
- 链接复制
- 表格
- 图片预览

## 结果

- 不依赖 HTML 注入。
- 容易按小程序交互定制。
- 解析能力有限，需要用测试锁定支持范围。
- 需求扩大时可重新评估成熟库。
